# DSA Grind Tracker

A modern, offline-first DSA roadmap and progress tracker designed for coding interview preparation, placements, internships, and top product-company interviews.

## Features

### Dual Roadmap System

The tracker contains two separate learning paths:

#### Basic Placement Roadmap

Focused on the most important interview patterns commonly asked in:

* Campus Placements
* Service-Based Companies
* Internship Interviews
* Entry-Level Software Engineering Roles

Topics include:

* Arrays & Hashing
* Strings
* Two Pointers
* Sliding Window
* Linked Lists
* Stack & Queue
* Binary Search
* Trees
* BST & Trie
* Heap
* Graphs
* Dynamic Programming
* Greedy
* Sorting
* Design Problems
* Math & Bit Manipulation

## Core Features

### Progress Tracking

* Mark problems as solved
* Automatically save progress using Local Storage
* Track topic-wise completion percentage
* View overall roadmap completion

### Search Functionality

* Search questions instantly
* Find problems across all topics
* Filter by keyword

### Difficulty Filters

Filter problems by:

* Easy
* Medium
* Hard
* Solved
* Unsolved

### Topic Progress

Every topic displays:

* Solved Count
* Total Questions
* Completion Percentage
* Visual Progress Bar

### Direct LeetCode Access

Every problem contains a direct link to its LeetCode page.

### Responsive Design

Works on:

* Desktop
* Laptop
* Tablet
* Mobile Devices

### Offline Support

Since all data is stored locally in the browser:

* No backend required
* No database required
* No account required
* Works completely offline

## Progress Persistence

Progress is automatically saved in Local Storage.

Data remains available even after:

* Browser refresh
* System restart
* Closing and reopening the website

unless Local Storage is manually cleared.
